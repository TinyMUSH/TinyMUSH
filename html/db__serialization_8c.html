Detailed Description

Note
Why not just write the attributes individually to disk? When synchronous writes are expensive, coalescing many attribute writes into a single operation yields a big speedup, especially for flatfile import/export. Sequential reads of a bundled object also get an implicit prefetch win.

While this was written decades ago, even an unoptimized sequence would likely be acceptable on modern hardware. If you want to feel the pain, dust off a retro box and benchmark it yourself.
