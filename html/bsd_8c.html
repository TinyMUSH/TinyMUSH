int process_input ( DESC * d)

Process input from a socket.

Parameters
dSocket description
Returns
int

Filter out problematic control characters at the network level. If we receive CTRL+C (0x03), close the connection gracefully.

If CTRL+C detected, close connection gracefully. Some telnet clients send CTRL+C which can cause desynchronization.

Filter out other dangerous control characters

Allow safe characters:

  • Printable ASCII (0x20-0x7E)
  • Essential whitespace:
    , \r, \t
  • ANSI ESC (0x1B)
  • DEL (0x7F) for backspace
  • BEEP (0x07) Reject all other control characters

◆ set_signals()

void set_signals ( void )

Set the signals handlers.

We have to reset our signal mask, because of the possibility that we triggered a restart on a SIGUSR1. If we did so, then the signal became blocked, and stays blocked, since control never returns to the caller; i.e., further attempts to send a SIGUSR1 would fail.
